设置页不要做成弹窗了，也做成单独的页面。
整体分为五个部分：通用设置、文本模型配置、生图模型配置、文件解析配置、关于。

# 通用配置
以后会放主题色、样式布局等设置项，现在留空即可。

# 文本模型配置
文本模型和之前一样，只支持OpenAI like 接口，用户输入base_url和key，可以获取可用模型，也能自己录入模型名称

# 生图模型
由于各家生图模型接口不统一，要先选择一个服务提供商，目前仅支持火山方舟和google ai studio两种方式，然后录入key，获取可用模型，也能自己录入模型名称

# 文件解析配置
文件解析是一个下拉框，包括本地解析、MinerU-精准解析 API、MinerU-Agent 轻量解析 API。

如果选择MinerU-精准解析 API，需要配置一个minerU的token。

配置页上显示如下对比表

| 对比维度       | 🎯 精准解析 API                                           | ⚡ Agent 轻量解析 API    | 本地解析   |
| :------------- | :------------------------------------------------------- | :---------------------- | ---------- |
| 是否需要 Token | 需要                                                   | 无需（IP 限频）       | ✅无任何限制 |
| 解析速度       | 慢 | 中等 |✅快|
| 支持文件格式   | pdf、jpeg、png、docx | pdf、jpeg、png、docx | ✅pdf、jpeg、png、docx、doc、wps、ofd |
| 文件大小限制   | ≤ 200MB                                                  | ≤ 10MB                  | ✅无限制 |
| 页数限制       | ≤ 200 页                                                 | ≤ 20 页                 | ✅无限制 |
| 解析质量 | 高                                      | 中               | ✅高 |
| 扫描件支持  | ✅支持 | 质量差 | ❌不支持 |
| 说明     |                                       |      | 招标文件大多数是word或word导出的带文字层的PDF，本地解析可以适应95%以上的情况，如果解析失败，可以尝试MinerU的精准解析API |
