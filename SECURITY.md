目前仅维护最新版本，建议始终使用最新版本。

## 报告安全问题

如果你发现安全漏洞或敏感问题，请不要公开提交 Issue。

请通过邮件反馈：

```text
fb208@outlook.com
````

反馈时建议说明：

* 问题描述
* 复现步骤
* 影响范围
* 修复建议（如有）

## 敏感信息提醒

请不要在 Issue、PR 或代码中提交以下内容：

* API Key
* Token
* 账号密码
* 未脱敏的招投标文件
* 客户资料或内部文档
