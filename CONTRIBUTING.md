
# 贡献指南

感谢你关注易标 AI（OpenBidKit_Yibiao）。

本项目是一个开源免费的 AI 标书编写工具，欢迎提交问题反馈、功能建议、文档优化和代码贡献。

## 支持项目

如果本项目对您有帮助，请礼貌地点一个 Star，这是对项目持续维护最大的支持。

## 反馈问题

如果你在使用过程中遇到问题，请优先提交 Issue。

提交 Issue 时，建议尽量说明：

- 操作系统
- 软件版本
- 问题描述
- 复现步骤
- 报错信息或截图

这样可以帮助我们更快定位和解决问题。

## 功能建议

如果你有新的功能想法，也欢迎通过 Issue 提出。

建议说明：

- 使用场景
- 当前遇到的问题
- 期望实现的效果

## 代码贡献

欢迎通过 Pull Request 参与项目开发。

提交 PR 前，请注意：

- 代码能够正常运行
- 不提交无关修改
- 不提交 API Key、Token、账号密码等敏感信息
- 尽量保持代码风格与现有项目一致

PR 描述建议包含：

- 修改内容
- 解决的问题
- 测试方式

## 本地开发

请先 Fork 本仓库，然后克隆到本地：

```bash
git clone https://github.com/你的用户名/OpenBidKit_Yibiao.git
````

安装依赖：

```bash
npm install
```

启动开发环境：

```bash
npm run dev
```

具体命令请以 README 中的说明为准。
